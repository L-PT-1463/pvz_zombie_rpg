# PvZ ZPG

Welcome to **PvZ ZPG - The Zombie RPG**, the name is a work in progress.

**PvZ ZPG** or Zombie Playing Game is a small roguelite game set in the PvZ universe, where you play as a zombie through various turned-base fights in a dungeon crawler-like world map, heavily inspired by Stray Fawn Studios' _Dungeon Clawler_.

## General Plans

This game is planned to include themed regions based on the worlds of the PvZ franchise, all with unique plant pools, loot and bosses. **PvZ ZPG** will also include a very important minion spawning mechanic, evoking the _I, Zombie_ minigame, one of the main inspirations.

## Disclaimer

I have no prior knowledge of coding and am doing this entirely to experiment with game development.

This is a small passion project so don't expect much progress, but still feedback and suggestions are greatly appreciated.

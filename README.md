# PvZ ZPG

Welcome to **PvZ ZPG - The Zombie RPG**, the name is a work in progress.

**PvZ ZPG** or Zombie Playing Game is a small roguelite game taking elements from all around the Plants vs Zombies franchise, where you play as a zombie through various turned-base fights in a dungeon crawler-like world map, heavily inspired by Stray Fawn Studios' _Dungeon Clawler_.

## General Plans

This game is planned to include themed regions based on the worlds of the PvZ franchise, all with unique plant pools, loot and bosses. **PvZ ZPG** will also include a very important minion spawning mechanic, evoking the _I, Zombie_ minigame, one of the main inspirations.

## Disclaimer

We're learning as we go - the Rogue Garden universe is our first major game development project and we have no real prior knowledge of coding, art and game design, animation and everything a video game might need.

This is a small passion project so don't expect much progress, but feedback and suggestions are greatly appreciated, so stay tuned for new updates. All the files we use will always be available on GitHub, so if you're intersted have a look and tell us what you think.
